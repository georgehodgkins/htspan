# htspan

## Dependencies

* gcc >= 4.8
* gsl >= 2.4.1

### Included dependencies

* htslib >= 1.8
* mlat >= 0.1

### Optional dependencies for testing and development

* samtools >= 1.8
* python3 >= 3.6
* boost >= 1.69
* bcftools >= 1.3.1

## Install

```{bash}
git clone https://github.com/djhshih/htspan
git submodule update --init --recursive
make
```
